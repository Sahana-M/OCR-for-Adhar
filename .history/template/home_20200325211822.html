Actyv ocr : Version.1

This API is designed to extract general fields from Adhar card

It accepts 2 types of document

  • Adhar front image
  • Adhar back image

    • A general field for uploaded Adhar front image includes,

  • Name
  • DOB
  • Gender
  • Father/Husband name
  • Age
  • Adhar number
  • Document identified

    • A general field for uploaded Adhar back image includes,

  • Address
  • Pincode
  • Document identified

    • Request format

  • Request - POST
  • Request URL - https://custom-ocr.herokuapp.com/get_adhar
  • In Body -> raw format -> JSON request
  • Example :- { "file" : "https://sankara-patients-aadhar.s3.amazonaws.com/8765510951%2F7c1f8d87-79b4-4e93-b503-0d5b04a159c7.jpg" }
